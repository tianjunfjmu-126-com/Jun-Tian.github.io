# 田俊 · 个人学术主页

本项目是田俊教授的个人学术网页，采用简洁、沉稳的学术风格，适用于 GitHub Pages 或个人学术展示。页面清晰展示了田俊教授的个人简介、研究方向、学术成果与荣誉。

## ✨ 页面特性

- **学术信息结构化**：包含个人简介、学术任职、主要荣誉、代表性论文、科研项目、专著教材及参考资料等模块。
- **视觉化身份标签**：使用徽章和关键词标签突出显示研究方向、职称与学会任职。
- **图片展示支持**：头部预留了圆形图片占位区，方便插入个人照片或学术相关图片。
- **响应式设计**：在桌面和移动设备上均能良好显示，信息层级清晰。
- **基于可靠数据**：内容整理自百度百科及公开学术记录，并标注了参考资料。
